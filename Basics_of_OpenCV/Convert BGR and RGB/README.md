# Convert BRG & RGB

* This Code converts a BGR image to RGB & vice versa.
* You can also [click here](https://github.com/tb-rules10/CV-Zone/blob/branch-1/Basics_of_OpenCV/Convert%20BGR%20and%20RGB/Code.py) to check the code.
